# 🌐 The Complete Web Development Bootcamp — Front-End

Este repositório reúne todos os projetos e exercícios desenvolvidos durante o curso **The Complete Web Development Bootcamp**, ministrado pela **Dr. Angela Yu** na Udemy.

 O objetivo é documentar minha jornada de aprendizado em **Desenvolvimento Web Front-End**, passando pelos fundamentos essenciais até projetos mais completos, incluindo HTML, CSS, JavaScript e outras tecnologias modernas utilizadas no dia a dia do desenvolvedor web.

Este repositório também funciona como um **portfólio de estudos**, sendo atualizado continuamente conforme avanço no curso.

## Projetos Desenvolvidos
### [PortifolioHTML](https://guilhermealves58.github.io/Curriculo-Web/)
Site de portfólio desenvolvido **exclusivamente com HTML**, com o objetivo de demonstrar tanto o domínio da estrutura e semântica da linguagem quanto minhas habilidades pessoais e profissionais.

### [Portifolio-Html-Css-Bootstrap](https://guilhermealves58.github.io/Portifolio-website-Html-Css/)
Site de portfólio desenvolvido com **HTML, CSS e Bootstrap**, representando uma evolução do projeto anterior feito apenas com HTML puro.  
Neste projeto, apliquei estilização com CSS e utilizei o Bootstrap para criar um layout mais organizado, responsivo e visualmente agradável.

### [Web-blog](https://github.com/GuilhermeAlves58/Web-blog-application)
O principal objetivo foi praticar a construção de um CRUD completo no servidor, trabalhando com rotas, renderização dinâmica, formulários e manipulação de dados em memória, sem uso de banco de dados.
A proposta do projeto é permitir que o usuário crie, visualize e edite posts de forma simples, com uma interface limpa e responsiva.

APIs e consumo de dados externos

JSON e manipulação de dados

Deploy de projetos (GitHub Pages, Render, etc.)

Boas práticas de organização e estruturação de código

Projetos completos integrando HTML + CSS + JavaScript
